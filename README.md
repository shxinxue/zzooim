## 一、Z站是什么？

**Z站（zzoo.im）是一个虚拟资源爱好者的交流社区。**

我们做的事情很简单：

*   **发现资源**：力求穷尽全球可以免费获取、试用的云服务器、云电脑、云手机、住宅代理ip、AI工具、API接口等……
*   **深度测评**：每一个站点、平台、资源，我们都会人工逐一测试，力求在全面的同时也够深刻。
*   **分享经验**：一个人的能力终究有限，Z站社区让天南海北的小伙伴共同分享交流关于虚拟资源的一切。
*   **激励活跃**：传统的垂直虚拟资源社区，几乎没有一个社区可以让你的评论、发帖等行为产生价值，而Z站不同，我们设置了Z币系统，虽然不能直接变现，但是也尽可能的让你兑换一切你所需的虚拟资源，包括但不限于云服务器、域名、云手机、云电脑、住宅代理IP等资源。

**一句话：来 Z站，理论上可以白嫖任何虚拟资源。**

## 二、Z站有哪些板块？

### 1. 社区首页
所有用户发布的帖子列表。你可以按分类筛选（服务器、域名、AI工具、云电脑等），也可以按“新贴”、“新评”或“热门”排序。

### 2. 深度测评
**这是 Z站 的核心内容板块。**

*   不再局限于站长发文，**任何用户都可以投稿深度测评**
*   内容方向：云服务器实测、住宅代理平台实测、云电脑测评、避坑分析等
*   投稿审核通过后，每篇文章可获得**至少 500 Z币**奖励
*   优质内容会从社区精选推荐到本栏目，也可独立投稿

> 社区 vs 深度测评：社区是“即时讨论”，深度测评是“沉淀精华”。

### 3. 网址导航
全球虚拟资源的一站式导航。免费服务器、云电脑、在线工具、白嫖福利……你想要的，这里都有。所有链接均经过人工验证，采用Z币激励方式保证网站及其活动的时效性。

在这里我们也引入了z币系统，第一次实现了在导航站上传网址，反馈网址的时效性、举报违法违规网站等也可获得一定数量的z币。

### 4. 在线工具（自研）
Z站 自研的免费实用在线工具，目标是涵盖所有虚拟资源类目。包括但不限于：

*   WHOIS查询
*   IP纯净度检测
*   域名比价、服务器等全球任意虚拟资源一键实时比价
*   免费图床
*   网页版多邮箱账户管理器，支持接入任意第三方大厂邮箱，无需切换账户收信（类似网易邮箱大师）。
*   图像压缩、文件转换、icon图标生成等小工具
*   谷歌邮箱、outlook邮箱等邮箱前缀可用性检测工具等一切你所需的工具

更多工具持续开发中。

### 5. 软件镜像
常用软件、工具、脚本的镜像下载服务。国内加速，方便快捷。

### 6. 任务福利
完成任务赚取 Z币的中心：

*   **新手任务**：注册、发首帖、设置头像/昵称/签名
*   **日常任务**：签到、发帖、评论
*   **挑战任务**：投稿测评、反馈建议、举报违规

### 7. 排行榜
社区活跃度榜单：

*   Z币榜
*   经验榜
*   签到榜
*   发帖榜
*   回复榜
*   后续还会逐步完善

## 三、Z币体系

### 1. Z币是什么？
Z币是 Z站 推出的**站内虚拟积分**，不具备实际货币价值，不可兑换人民币。它用于激励用户在社区内的活跃行为。

### 2. Z币用途
兑换 Z币商城内的虚拟资源，包括但不限于：

*   本站会员（时长制）
*   谷歌邮箱账户
*   fj账号
*   云服务器
*   视频网站会员
*   其他等用户所需的资源，可以根据用户需求补充库存。

> **Z币不支持提现、充值、转账，仅限站内使用。**

## 五、发帖与评论规则

### 1. 发帖规则

| 项目 | 说明 |
| :--- | :--- |
| **允许内容** | 资源分享、测评、教程、避坑经验、技术讨论、福利情报 |
| **禁止内容** | 违法信息、色情、赌博、暴力、政治、未成年等违法、敏感话题 |
| **禁止行为** | 人身攻击、地域歧视、恶意刷屏、垃圾广告 |

### 2. 评论规则

| 项目 | 说明 |
| :--- | :--- |
| **鼓励行为** | 友善讨论、技术交流、补充信息 |
| **禁止行为** | 辱骂、引战、刷屏、无意义回复（如“顶”、“111”） |
